Design Reasoning

ทำไมต้องลดสี?

Meditation site ควรให้ความรู้สึก "สงบ" — ยิ่งมีสีน้อย ยิ่งรู้สึกโล่ง สะอาดตา ไม่รบกวนสายตา เหมือนเดินเข้าวัดแล้วเจอพื้นที่โปร่ง ไม่ใช่ตลาดนัด

ทำไมเก็บ Gold ไว้?

Gold (#D4A847) ยังมีประโยชน์สำหรับ chip "มีค่าใช้จ่าย" — สร้าง contrast ชัดเจนกับ green chips (ฟรี) ช่วยให้ user scan ได้เร็ว แต่ไม่ควรใช้ที่อื่นนอกจากนี้

Secondary Sage (#A8D5BA) หายไปไหน?

ถูกแทนที่ด้วย Primary Tint (#EFF8F2) ซึ่งทำหน้าที่เดียวกัน (chip bg, hover state, highlight) แต่ subtle กว่า และ derive มาจาก primary green ได้เลย ไม่ต้องจำสี extra

Gradient ยังใช้ได้ไหม?

gradient-primary (green → sage → gold) ควรลดเหลือ green → green-light ก็พอ ไม่ต้อง 3 จุด — เรียบง่ายกว่า ดู premium กว่า
